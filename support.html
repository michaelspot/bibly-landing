Questions fréquentes

Comment annuler mon abonnement ?

Les abonnements sont gérés directement par l'App Store (iOS) ou le Google Play Store (Android). Pour annuler, rendez-vous dans les paramètres de votre compte sur le store correspondant, puis dans la section "Abonnements".

Comment restaurer mes achats ?

Dans l'application, va dans Paramètres puis appuie sur "Restaurer les achats". Assure-toi d'être connecté avec le même compte Apple ID ou Google que celui utilisé lors de l'achat initial.

Les widgets ne s'affichent pas correctement, que faire ?

Essaie de supprimer le widget puis de le rajouter. Si le problème persiste, redémarre ton appareil. Sur iOS, assure-toi également que l'actualisation en arrière-plan est activée pour Bibly dans les réglages.

Comment supprimer mes données ?

La majorité de tes données sont stockées localement sur ton appareil. Pour les supprimer, il te suffit de désinstaller l'application. Pour toute demande concernant des données détenues par nos partenaires, contacte-nous par email.

Je ne reçois pas les notifications, pourquoi ?

Vérifie que les notifications sont bien activées pour Bibly dans les réglages de ton téléphone. Assure-toi également d'avoir configuré tes préférences de notifications dans l'application (Paramètres > Notifications).

L'application plante ou ne fonctionne pas correctement

Assure-toi d'avoir la dernière version de l'application installée. Si le problème persiste, essaie de désinstaller puis réinstaller l'application. N'hésite pas à nous contacter en décrivant le problème rencontré.
